# Discovery and Learning with Big Data project
Project Summary

This project demonstrates the application of machine learning across multiple business and analytical scenarios. It consists of six major parts covering AI strategy development, data preprocessing, supervised learning, unsupervised learning, and model comparison.

Part I: Machine Learning Strategy for a Cancer Center

A comprehensive machine learning implementation strategy was designed for a hypothetical Denton Cancer Center (DCC). The plan includes data infrastructure modernization, cloud-based data architecture, MLOps, AI governance, staff training, ethical AI practices, and deployment of predictive models for oncology, radiology, pathology, operations, and research. The CRISP-DM framework was proposed as the standard project lifecycle.

Part II: Data Preprocessing

A shelter animal dataset was cleaned by handling missing values, converting categorical variables to numeric form, analyzing correlations, and selecting relevant features. A cleaned dataset was produced for future modeling.

Part III: Abalone Age Prediction (Regression)

Machine learning models were developed to predict abalone age using physical measurements such as length, diameter, height, and weight. Linear Regression and Decision Tree Regression were compared. Linear Regression achieved a higher R² (0.49) and was selected as the preferred model.

Part IV: Income Classification

Using demographic and employment information, Logistic Regression and K-Nearest Neighbors (KNN) were trained to classify individuals as earning above or below $50,000 annually. KNN outperformed Logistic Regression, achieving approximately 84% accuracy.

Part V: Car Market Segmentation (Clustering)

K-Means clustering was applied to a car evaluation dataset. The Elbow Method identified three optimal clusters, enabling segmentation of vehicles based on price, maintenance cost, passenger capacity, safety, and luggage size.

Part VI: Model Comparison

Regression models including Linear Regression, Decision Tree, and Random Forest were compared for abalone age prediction. Random Forest achieved the best performance (R² ≈ 0.55), demonstrating the strongest predictive capability among the evaluated models.
Overall Conclusion

The project demonstrates the complete machine learning lifecycle—from organizational AI strategy and data preparation to regression, classification, clustering, model evaluation, and deployment considerations—while highlighting the strengths and limitations of different machine learning approaches in real-world applications.
